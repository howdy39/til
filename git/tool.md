# Git-Completion

## ファイル落とす
```
curl https://raw.githubusercontent.com/git/git/master/contrib/completion/git-completion.bash  > ~/.git-completion.bash
```

## .bash_profileに設定を追加
```
source ~/.git-completion.bash
```
